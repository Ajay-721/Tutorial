# Prompt Templates

Use these templates to interact with Claude Code for bug reports and feature requests.

## How to use
1. Copy the template below
2. Fill in your details
3. Paste into Claude Code

---

## Template 1 — Bug Report

```
Acting as the Code Reviewer agent in CLAUDE.md, handle this bug report:

**Title:** [Short description of the bug]
**Priority:** [Critical / High / Medium / Low]
**What happens:** [Describe what goes wrong]
**What should happen:** [Describe expected behaviour]
**Steps to reproduce:**
1. [Step 1]
2. [Step 2]
3. [Step 3]
**Affected users:** [All users / Specific group]
```

---

## Template 2 — New Feature Request

```
Acting as the Code Reviewer agent in CLAUDE.md, handle this feature request:

**Title:** [Short name for the feature]
**Who will use it:** [Regular users / Admin / Both]
**Why we need it:** [Problem it solves or value it adds]
**Out of scope:** [What is NOT included in this version]
**Priority:** [High / Medium / Low]
**UX notes:** [Any design or behaviour preferences, or write SKIP]
```
