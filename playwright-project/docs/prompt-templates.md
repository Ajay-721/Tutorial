# Prompt Templates

## How to raise a bug

Just type one of these in Claude Code:

- `new bug`
- `raise bug`
- `report bug`

Claude Code will interview you with 6 questions then run the full bug workflow automatically.

---

## How to request a feature

Just type one of these in Claude Code:

- `new feature`
- `add feature`
- `feature request`

Claude Code will interview you with 6 questions then run the full feature workflow automatically.
